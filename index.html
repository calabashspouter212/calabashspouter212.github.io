🎞️ scriptbreak - Organize your screenplay into AI assets

scriptbreak processes screenplays into production-ready documents. It extracts scenes, elements, and character bibles. It creates shot lists and timelines. You export finished prompt packs for use with any image or video generation tool. It functions entirely on your computer without the need for API keys.

🚀 Getting Started

Installation requires only a few minutes. This software runs as a standard desktop application on Windows 10 and Windows 11. Follow these steps to set up the environment.

📥 Download and Setup

  1. Visit the official download page.
  2. Select the file ending in .exe for your Windows system.
  3. Open the downloaded folder and run the installer.
  4. Follow the prompts on your screen to complete the installation.
  5. Launch the program from your desktop shortcut or start menu.

📋 How to Use the Studio

The studio interface handles your script files directly. You do not need technical knowledge to manage your projects.

1. Import Your Script

The software supports standard formats like Fountain and Final Draft. Click the Import button and select your file. The engine reads your text and identifies character names, locations, and narrative beats automatically.

2. Manage Elements

Review the list of characters and scenes. You can edit names or combine locations to refine your production bible. These settings inform how the software writes your prompts.

3. Generate Visual Assets

Choose your target creative tool. You can adjust settings to match the aesthetic requirements of your video or image generator. The software transforms the script data into structured prompt packs.

4. Export Your Data

Click the export button to save your prompt packs. These files contain all data necessary for your external AI tools. Since the process stays on your local machine, your data remains private.

💡 Frequently Asked Questions

Does this require an internet connection?

The tool works offline. You do not need an account or an API key for any part of the process.

Are my scripts kept private?

Yes. All processing happens on your local device. Your files never leave your computer.

Which script formats do you support?

The software supports Fountain and Final Draft files. Plain text files also work well if they contain standard screenplay headers.
